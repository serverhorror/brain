#+title: Go

* Go - Customizing Go Binaries with Build Tags


- [[https://www.digitalocean.com/community/tutorials/customizing-go-binaries-with-build-tags][Customizing Go Binaries with Build Tags (digitalocean.com)]]

In your source code put this as the **first** line:

#+begin_src go
  // +build tag_name
#+end_src

or (to include):

#+begin_src go
  //go:build tag_name
#+end_src

alternatively (to exclude):

#+begin_src go
  // +build !tag_name
#+end_src

Then build with:

#+begin_src go
  go build -tags tag_name
#+end_src
