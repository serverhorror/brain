* [[docker.rst]] mounts =wayland-0= on Windows

Recent versions of the _[[https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers][Dev Containers ms-vscode-remote.remote-containers]]_ extension introduce a new setting.
This setting tries to detect whether it can mount a socket "=wayland-0=".
This detection fails with =fedoraremix=.
The setting below disables this detection and removes the automatic mount from the docker command that starts the container.

#+begin_src json
{
  "dev.containers.mountWaylandSocket": false
}
#+end_src
